# Ouroboros System - Scripts

Utility scripts for common tasks.

## Available Scripts

### Startup Scripts
- `start.sh` / `start.ps1` - Start the system with full setup
- `verify.sh` - Quick verification wrapper
- `generate.sh` - Generator wrapper

### Testing
- `test.sh` - Run tests with coverage

### Deployment
- `deploy.sh` - Deploy to Kubernetes

## Usage

### Linux/Mac
```bash
chmod +x scripts/*.sh
./scripts/start.sh
./scripts/test.sh
./scripts/deploy.sh production
```

### Windows
```powershell
.\scripts\start.ps1
```
